# Zequent Framework Edge SDK
The Edge SDK is for implementing Edge Clients to Zequent Framework Ecosystem to be used from the same interface and same methods from client side.


## Tech Specs

* Java: 25
* Maven: 3.9.9

## Core Concepts

* Edge Adapter
* Live Data
* Connector


### Edge Adapter 
The Edge Adapter Interface defines the actions and methods that are available. 
Every Adapter can make his own custom implementation.


### Quick Start

```xml
<dependency>
  <groupId>com.zequent.framework.edge.sdk</groupId>
  <artifactId>java-edge-sdk</artifactId>
  <version>1.0-SNAPSHOT</version>
</dependency>
```
